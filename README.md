# 📸 Project Overview

NexusNews transforms global news into an interactive intelligence command center inspired by military cyber operations interfaces.

The platform combines:

* Real-time news aggregation
* AI-generated geopolitical analysis
* Global threat visualization
* Crisis intelligence systems
* Advanced analytics dashboards

Designed with a futuristic classified-operations aesthetic.

# 🚀 Local Development Setup

## 1. Clone Repository

```bash
git clone https://github.com/sameer-sde/nexusnews.git
cd nexusnews
```

---

## 2. Install Dependencies

```bash
npm install
```

---

## 3. Configure Environment Variables

Create `.env.local`

```bash
cp .env.local.example .env.local
```

Add your API keys:

```env
VITE_ANTHROPIC_API_KEY=your_anthropic_api_key
VITE_NEWS_API_KEY=your_newsapi_key
```

---

# 🚢 Deploy to Vercel

## Build Project

```bash
npm run build
```

---

## Deploy Steps

1. Push project to GitHub
2. Import repository into Vercel
3. Add environment variables in:

```text
Project Settings → Environment Variables
```

4. Redeploy application

---

# 📁 Environment Variables Example

`.env.local.example`

```env
VITE_ANTHROPIC_API_KEY=your_api_key_here
VITE_NEWS_API_KEY=your_api_key_here
```

⚠️ Never upload real API keys to GitHub.

* avoid exposing API keys in frontend environments,
* use secure server-side authentication layers.
