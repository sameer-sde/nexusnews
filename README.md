# NexusNews — AI War Room Dashboard

A cinematic AI-powered global intelligence dashboard featuring real-time geopolitical monitoring, live news tracking, AI threat analysis, and strategic risk visualization.

## 📸 Overview

NexusNews transforms global news into an interactive intelligence command center.
Built with React + Vite, it combines AI analysis, geopolitical tracking, and modern data visualization into a futuristic dashboard experience.

# 🚀 Local Setup

## 1. Clone Repository

```bash
git clone https://github.com/sameer-sde/nexusnews.git
cd nexusnews
```

## 2. Install Dependencies

```bash
npm install
```

## 3. Configure Environment Variables

Copy the example file:

```bash
cp .env.local.example .env.local
```

Open `.env.local` and add your API keys:

```env
VITE_ANTHROPIC_API_KEY=your_anthropic_api_key
VITE_NEWS_API_KEY=your_newsapi_key
```

### 🔑 Get API Keys

* Anthropic API: https://console.anthropic.com
* NewsAPI: https://newsapi.org/register

---

# ▶️ Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

---

# 🚢 Deploy to Vercel

Build the project:

```bash
npm run build
```

Then:

1. Push the project to GitHub
2. Import the repository into Vercel
3. Add environment variables in:

```text
Vercel Dashboard → Settings → Environment Variables
```

4. Redeploy the application

---

# 📁 Environment Variables

Example `.env.local.example`

```env
VITE_ANTHROPIC_API_KEY=your_api_key_here
VITE_NEWS_API_KEY=your_api_key_here
```

⚠️ Never commit real API keys to GitHub.
